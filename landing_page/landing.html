RemoteMe

The laptop-friendly workspace finder.

About

Remote Me is a web app that allows users to find, list, and rate areas in their city that are laptop-work friendly areas (such as cafes, bars & hotels). All locations will be listed on the interactive map.

Live Site

A live version of the web app is available via the links above at Heroku, as well as access to the open-source code hosted on Github.

Please Note: The live version available may not be up to date with the code on Github.
